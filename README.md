# ZEN-G E-Commerce Website

A modern, responsive e-commerce website built with React, Tailwind CSS, Material UI, and Redux.

## Features

- **Home Page**: Image slider with product cards
- **Collections Page**: Browse products by category and type (Basic, Core, Premium)
- **Product Detail Pages**: Filter products by size, color, and price
- **Shopping Cart**: Add, remove, and update quantities
- **Wishlist**: Save favorite products
- **Search**: Search products across the store
- **Contact Us**: Contact form with map integration
- **About Us**: Company information
- **Sign In/Login**: Phone number authentication

## Tech Stack

- **React 18** - UI library
- **React Router** - Navigation
- **Redux Toolkit** - State management
- **React Context** - Additional state management
- **Tailwind CSS** - Styling
- **Material UI** - UI components
- **Vite** - Build tool

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Project Structure

```
src/

├── Routes/          # Route configuration
├── Services/        # API services
```
